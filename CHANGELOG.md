# Changelog

## [1.0.0] - 2026-06-10

### Added
- Menú interactivo con diseño visual y arte ASCII arcoíris
- 3 modos predefinidos: Editar Timeline, Programar Cursor, Multimedia
- Modo Personalizado con configuración guiada paso a paso
- Editor directo de configuración keyd
- Comandos rápidos: `on`, `off`, `status`
- Banner personalizable via `banner.txt`
- Logs paso a paso con colores
- Fallback automático: `keyd reload` → `systemctl restart keyd`
- Detección de errores con mensajes de ayuda
- Soporte para cualquier teclado configurando `KEYBOARD_ID`
- Documentación profesional completa
- Licencia MIT

### Keyd
- keyd es el motor de reasignación de teclas: https://github.com/rvaiya/keyd
